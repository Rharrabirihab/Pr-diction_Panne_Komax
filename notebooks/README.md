# Prédiction des Pannes Komax — PFE SEWS CABIND MAROC

## Description
Application de prédiction des pannes pour les machines de coupe Komax
utilisant le Machine Learning.

## Modèles ML
- Modèle 1 : Prédiction du Type de Panne (Classification)
- Modèle 2 : Prédiction du Down Time (Régression)  
- Modèle 3 : Prédiction du Serial Device (Classification)
- Modèle 4 : Recommandation de l'Action (Classification)

## Installation
pip install -r requirements.txt

## Lancer l'app
streamlit run streamlit_app/app.py
